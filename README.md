# YouTube Transcript Extractor & Cleaner

A modern, fast Windows desktop application to fetch, clean, and format YouTube video transcripts. It processes raw transcripts, strips timestamps or bracketed annotations, collapses double spaces, and neatly formats them into readable paragraphs or lines. You can also export the cleaned transcripts directly as Text or PDF files.

![UI Preview](https://raw.githubusercontent.com/username/YouTubeTranscribeExtractor/main/preview.png) <!-- Replace with actual screenshot when uploaded -->

## Features
- **Fetch Transcripts Automatically**: Enter any YouTube Video URL or ID to pull the transcript instantly.
- **German/International Character Support**: Full Unicode support for German umlauts (`ä`, `ö`, `ü`, `ß`) and special characters.
- **Smart Formatting**: Removes timestamps and formatting, and rebuilds readable paragraph structure (paragraph breaks are automatically added where pauses are greater than 10 seconds).
- **Export Options**: Save cleaned transcripts directly as plain text (`.txt`) or format-wrapped `.pdf` documents.

## How to Install & Run
1. Go to the [Releases](https://github.com/username/YouTubeTranscribeExtractor/releases) page.
2. Download `YouTubeTranscribeExtractor.exe`.
3. Double-click to run! 

*Note: The application is fully self-contained. It embeds all necessary python-based transcription modules internally, so you do not need to install Python or any external packages on your computer.*
